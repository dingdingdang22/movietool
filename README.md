# 🎬 电影字幕智能分割工具 (Movie Smart Splitter)

这是一个基于 Python 和 FFmpeg 开发的自动化视频处理工具。它能够根据电影或视频的字幕文件（.srt），智能分析对话时间轴，**自动剔除长时间无对话的“静音”片段**，并按照用户指定的时长范围**将长视频切分为多个单集短视频**，同时自动重构对应的字幕文件。

非常适合用于影视解说、英语学习素材制作、长视频切片等场景。

## ✨ 核心特性

- ✂️ **智能静音剔除**：根据字幕时间轴自动识别并跳过连续无对话的冗余片段（如长镜头、片头片尾等）。
- 📦 **灵活分集切割**：支持设置最小和最大分集时长（如每集 5-10 分钟），工具会自动寻找合适的切分点进行无缝截断。
- 📝 **字幕自动重构**：切割后的每一集视频都会配套生成全新的字幕文件，时间轴自动从 `00:00:00` 重新对齐。
- 🌍 **双字幕支持**：支持同时导入主字幕和第二字幕（如中英双语），切割时同步处理两套字幕。
- 🚀 **极速无损/压缩处理**：
  - 默认使用 FFmpeg Stream Copy（流拷贝）技术，实现无损且极速的视频裁剪与合并。
  - 可选开启“网络串流优化压缩”模式（转码至 720p、H.264视频、AAC音频格式），大幅减小文件体积，完美兼容 iPhone/Safari 等移动端网页直接播放。
- 🖥️ **双交互模式**：
  - **GUI 图形界面**：直接双击运行，提供直观的弹窗引导，对零基础用户极其友好。
  - **CLI 命令行**：提供完善的参数支持，适合高级用户和自动化脚本调用。

## 🛠️ 环境依赖

在使用本工具之前，请确保您的系统中已安装以下依赖：

1. **Python 3.7+**
2. **FFmpeg**：
   - 请下载并安装 FFmpeg。
   - 确保 `ffmpeg` 命令已被添加到系统的环境变量（PATH）中，或者将 `ffmpeg.exe` 放置在与程序同级的目录下。
3. **Python 依赖库**：
   在命令行中执行以下命令安装所需依赖包：
   ```bash
   pip install pysrt
   ```
   *(注：图形界面使用的是 Python 内置的 Tkinter 库，通常无需额外安装。)*

## 🚀 使用指南

### 方式一：小白模式（图形化界面）

直接在文件夹中**双击运行** `main.py`（或者通过终端无参数运行 `python main.py`）。

程序会自动检测到您没有传入参数，并弹出图形化窗口：
1. 依次选择原始视频文件。
2. 选择对应的字幕文件（主字幕）。
3. 选择第二字幕文件（没有可直接点取消）。
4. 按照提示输入**静音剔除阈值**、**最小分集时长**和**最大分集时长**。
5. 选择是否开启网络压缩优化。

随后程序将自动开始在控制台输出处理进度，完成后可在上一级的 `output` 文件夹中找到生成的分集视频和字幕。

### 方式二：极客模式（命令行工具）

在终端或命令行中执行 `main.py`，并附带相关参数。

**基础用法：**
```bash
python main.py -v "movie.mp4" -s "movie_en.srt"
```

**进阶用法（双字幕、调整阈值、开启压缩）：**
```bash
python main.py -v "movie.mp4" -s "en.srt" -s2 "zh.srt" --silence-threshold 20.0 --min-minutes 3.0 --max-minutes 8.0 --compress
```

### ⚙️ 命令行参数说明

| 参数 | 简写 | 是否必填 | 默认值 | 描述 |
| :--- | :---: | :---: | :--- | :--- |
| `--video` | `-v` | ✅ 是 | 无 | 原始视频文件路径 (如 .mp4, .mkv) |
| `--subtitle` | `-s` | ✅ 是 | 无 | 原始主字幕文件路径 (.srt) |
| `--subtitle2` | `-s2`| ❌ 否 | `""` | 第二字幕文件路径 (.srt)，用于双字幕输出 |
| `--output` | `-o` | ❌ 否 | `../output` | 输出目录路径，最终会在该目录下自动生成一个同名文件夹 |
| `--silence-threshold`| 无 | ❌ 否 | `30.0` | 静音剔除阈值（秒）。连续无对话超过此时长将被剔除 |
| `--min-minutes` | 无 | ❌ 否 | `5.0` | 最小分集时长（分钟） |
| `--max-minutes` | 无 | ❌ 否 | `10.0` | 最大分集时长（分钟），需大于等于最小分集时长 |
| `--compress` | 无 | ❌ 否 | `False` | 是否开启视频压缩（720p）及网络串流优化 |

## 📂 目录与文件说明

- `main.py`：项目入口，负责启动双模式界面及模块调度。
- `timeline_analyzer.py`：时间轴分析器，负责解析字幕计算切割点与丢弃区。
- `subtitle_reconstructor.py`：字幕重构器，负责生成切分后归零对齐的新字幕。
- `video_processor.py`：视频处理器，负责封装 FFmpeg 极速裁剪与无缝合并底层逻辑。
- `video_splitter.py`：基础测试与开发原型脚本。

## ⚠️ 注意事项

- 源视频文件名及路径中建议**尽量避免使用复杂的特殊字符**，以防 FFmpeg 路径解析错误。
- 若开启了 `--compress`，处理时间会显著增长，具体取决于您的 CPU 性能。
- 本工具的核心逻辑严重依赖字幕文件的时间轴准确度，请确保**提供的字幕与视频时间是完全对齐同步的**。
