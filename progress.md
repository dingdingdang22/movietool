# 项目开发总控进度表 (Progress & Task Checklist)

## Phase 1: 时间轴分析模块 (Timeline Analyzer)
- [x] 1.1 定义核心数据结构 `TimeRange`, `DialogueInfo`, `SplitPlan`。
- [x] 1.2 实现 `pysrt` 字幕读取与解析，转换为 `DialogueInfo` 列表。
- [x] 1.3 开发静音检测算法：遍历相邻字幕计算时间差，标记长度大于 `SILENCE_THRESHOLD_SEC` 的剔除区间。
- [x] 1.4 开发基于弹性的切割点搜索算法：累加保留时间，当达到 `MIN_SEGMENT_MINUTES` 时，在随后的停顿中寻找最大间隙。
- [x] 1.5 强制兜底逻辑：若达到 `MAX_SEGMENT_MINUTES` 未找到理想断点，则强制切割，避免单集超时。
- [x] 1.6 组装 `SplitPlan` 列表输出，编写单元测试验证时间计算和切分逻辑的准确性。

## Phase 2: 字幕重构模块 (Subtitle Reconstructor)
- [x] 2.1 根据传入的 `SplitPlan` 的原视频保留区间，提取对应的字幕条目。
- [x] 2.2 实现时间戳重计算算法：扣除被剔除的静音片段耗时。
- [x] 2.3 实现分集基准归零：每集视频首句字幕的起始时间根据新时间轴相对归零（首集从头计算，后续集按各自0点计算）。
- [x] 2.4 利用 `pysrt` 将修改后的字幕对象输出分集的双语 `.srt` 文件（如 `[Name]_Part01.srt`）。
- [x] 2.5 测试验证：校验新生成的字幕时间轴是否有逻辑断层或负数。
- [x] 2.6 支持双字幕（主辅字幕）独立导入与同步重构，输出 `_sub1.srt` 和 `_sub2.srt`。

## Phase 3: 视频处理模块 (Video Processor / FFmpeg Wrapper)
- [x] 3.1 封装基础 FFmpeg 调用命令 (`subprocess.run`)。
- [x] 3.2 实现单段视频的高速流拷贝裁剪 (FFmpeg `-ss` 与 `-to` 参数，结合 `-c copy`)。
- [x] 3.3 实现同一集中多个零碎保留片段的无缝拼接 (FFmpeg `concat demuxer`，生成临时 `concat.txt` 并合并)。
- [x] 3.4 异常处理机制：捕获 FFmpeg 报错并进行终端日志输出。
- [x] 3.5 联调测试：视频与重构后字幕同轨播放（手动验证音画字同步情况）。
- [x] 3.6 增加视频压缩与网络串流优化：支持转码为 720p 分辨率、H.264 视频与 AAC 音频编码，保障 iPhone Safari 等移动设备网页端兼容性，并启用 Web 优化 (`-movflags +faststart`)。
