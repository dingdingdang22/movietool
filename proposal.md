# 电影字幕驱动的视频智能分割工具需求文档 (Proposal)

## 1. 项目背景与目标
本项目旨在开发一个基于 Python 的自动化视频处理脚本。该工具能够读取本地电影文件及其对应的外挂双语字幕文件，根据字幕时间轴将电影智能分割为多个约 5-10 分钟的片段，并自动剔除没有对话的长时间冗余片段，最终输出按序重命名且剧情连贯的视频片段与配套字幕。

## 2. 运行环境与依赖
* **操作系统**：Windows
* **核心编程语言**：Python 3.x
* **核心多媒体处理工具**：FFmpeg (需配置到系统环境变量)
* **Python 依赖库** (预期)：`pysrt` (用于解析字幕), `subprocess` (用于调用 FFmpeg)

## 3. 核心功能需求

### 3.1 视频与字幕解析
* **多格式支持**：兼容 FFmpeg 支持的主流视频格式（如 `.mp4`, `.mkv`, `.avi` 等）。
* **外挂字幕读取**：读取外部独立字幕文件（如 `.srt`），提取每句对话的开始时间和结束时间。

### 3.2 智能静音/无对话裁剪 (方案 B)
* **设定静音阈值**：设定一个无对话判定阈值（例如：连续 30 秒无字幕）。
* **静音剔除**：如果两句字幕之间的间隔超过该阈值，则将该无字幕片段标记为剔除区。较短的停顿（如几秒钟的呼吸或动作时间）将被保留，以防视频画面频繁跳跃（鬼畜），保证观影流畅度。

### 3.3 弹性剧情分割 (约 5-10 分钟)
* **场景连贯性**：程序将以 5-10 分钟为基准长度进行片段划分。
* **弹性切割点寻找**：在接近或达到 10 分钟时，程序不会在某句话说到一半时强制切断。相反，它会寻找字幕时间轴里的“最长停顿间隙”作为天然的场景切换点。如果为了保证一段完整对话不被切断，允许该片段时长适度弹性超时（例如延长到 11 或 12 分钟）。

### 3.4 字幕重构与双语保留
* **时间轴重计算**：由于视频经过了剔除和分割，程序必须对新生成的视频片段重新计算字幕时间轴，确保音画字完美同步。
* **双语字幕导出**：每个分割出的视频片段，均需配套生成对应的独立字幕文件，保留原有双语内容不变。

### 3.5 自动化重命名与输出
* **统一命名规范**：处理后的视频和字幕文件将自动按序排列。
  * 视频示例：`MovieName_Part01.mp4`, `MovieName_Part02.mp4`
  * 字幕示例：`MovieName_Part01.srt`, `MovieName_Part02.srt`
* **无损/快速切割**：在条件允许的情况下，尽量使用 FFmpeg 的 Stream Copy (流拷贝) 功能进行无损切割，以极大加快处理速度并保持原画质。

## 4. 后续开发规划
1. 编写 Python 脚本实现字幕文件(`.srt`)的解析与时间轴分段算法。
2. 编写 FFmpeg 调度模块，根据计算出的时间轴执行裁剪与拼接。
3. 对齐并输出新的字幕文件。
4. 进行系统级联调与测试。
